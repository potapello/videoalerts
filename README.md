# Videoalerts Gate

## КАК НАЧАТЬ

После скачивания релиза нужно настроить сервер.

1. Откройте **"options.cfg"**, тут можно настройть сервер. После любых изменений сервер нужно перезапускать.
2. Обязательно в **"options.cfg"** настроить поля **channelID** и **readGlobalChat**, что они значат написано в самом файле.
3. Попробуйте запустить **"videoalertsgate.exe"**. Может запуститься не с 1 раза, а только после разрешения пользования инетом.

Если сервер работает, нужно добавить клиент в ОБС.

1. Откройте **OBS**, добавьте источник **Browser** с такой ссылкой = `http://localhost:3000`. Разместите источник в иерархии выше захвата игры/экрана, чтобы он был виден.
2. Укажите размер 1920x1080 (дефолтный). Можно использовать другой, но сначала его нужно указать в **"options.cfg"**.
3. Если виден зелёный фон, добавьте **Color Key** фильтр для **Browser**, измените цвет на `#00ff00` или `Green`, фон должен исчезнуть.
5. Убедитесь, что клиент подключился к серверу (обновите **Browser**, будет сообщение), попробуйте ввести в чате вашего стрима команду "!ma ". (гайд по команде ниже)
6. Видеоалерт должен появиться в **Browser** в ОБС.
7. Рекомендую вывести звук из **Browser** в отдельный микшер и выставить что-то около **-10dB**. (чтобы слышать всем, выставьте режим для микшера "Прослушивание и вывод")
8. В папке "views" можно найти тестер **"tester.html"**, в нём можно контролить видеоалерты.

Сначала запускаете сервер, а уже после запускаете ОБС! Иначе после запуска сервера нужно обновить **Browser** в ОБС, чтобы тот подключился в серверу!

## Как использовать команду `!ma <url> [modifier]`

`<url>` обязательно. Сссылка на видео для алерта, поддерживаются следующие:
- Memealerts `cdns.memealerts.com`
- Youtube videos & shorts `youtube.com, youtu.be`
- Tiktok videos & images `tiktok.com`

Можно разрешить любые прямые ссылки на видео, выставьте **allowUnknownSourceVideos=true** в файле **options.cfg**.

`[modifier]` опционально. Пользователи могут применять моды для видеоалертов:
- `pos [x] [y]` указать позицию
- `rotate` вращать видео
- `faster`, `slower` выставить скорость проигрывания **(x1.5 and x0.7)**
- `speed [x]` выставить свою скорость проигрывания в диапазоне **[0.25; 3.0]**
- `party` смена цвета по hue кругу и тряска
- `cursed` тёмное черно-белое видео
- `wide`, `tall` делает видео широким/высоким
- `invert` негатив цветов
